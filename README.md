## 📌 Project Title:

Operating System Simulation

## 🎯 Objective

To simulate basic functions of an Operating System like:

- Process scheduling
- CPU management
- Memory allocation
- Simple task execution

## 🧠 Concept

An Operating System (OS) is a software that manages hardware and software resources.

In this project, we simulate:

- How processes are created
- How CPU gives time to each process
- How tasks run one by one
- import time
  
## ⚙️ Features

- Simulates Round Robin CPU Scheduling
- Shows process execution order
- Handles multiple processes
- Demonstrates time-sharing concept

- OS learning projects
- College assignments
- Interview preparation
